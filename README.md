# Meta-Prompt Generator

This project is a web-based tool to generate meta-prompts for use in various tasks.

## Features
- Select roles, protocols, and architectures.
- Automatically generate a meta-prompt based on the selections.
- Simple and intuitive web-based interface.

## Files
- **data.json**: Contains roles, protocols, architectures, and recommended tools.
- **app.py**: Backend implemented with Flask for meta-prompt generation.
- **index.html**: Frontend interface for user input and prompt display.

## How to Run
1. Install dependencies:
    ```bash
    pip install flask
    ```
2. Start the backend server:
    ```bash
    python app.py
    ```
3. Open the `index.html` file in a browser.
